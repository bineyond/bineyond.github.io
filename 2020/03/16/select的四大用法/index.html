select 跟 switch 有个共同特性就是都通过 case 的方式来处理，但是 select 跟 switch 处理的事情完全不同，也完全不相容。

switch

特性:

  • 1.各种类型及型别操作，接口 interface{} 型别判断 variable.(type)
