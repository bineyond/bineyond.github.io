在Mac上安装db2 driver

发表于 2020-12-28 更新于 2021-11-23 本文字数： 359 阅读时长 ≈ 1 分钟

选择版本

此处选择

11 版本开始有 MacOS 版本支持，需下载11版本。
选择 【IBM Data Server Driver Package】

安装或更新

  • 通过双击 v11.5.5_macos_dsdriver.dmg 文件，装载磁盘映像
  • 这将打开新的 Finder 窗口，其中会显示该磁盘映像的内容
  • 如果 Finder 窗口未打开，请双击桌面上的 dsvriver 图标
  • 在 Finder 窗口中，双击 installDSDriver.sh 文件
  • 驱动程序包将安装在以下缺省位置：/Applications/dsdriver
  • 配置环境变量 vi ~/.zshrc

配置

  • 配置本地环境以连接到 Db2 数据库
# db2 # MacOS
